# How We Manage AI Tools

## How we manage AI tools at Danish Agro

We don't maintain a "positive list" of approved AI tools. Instead, we take a practical approach:

**What we do:**

- Monitor all AI tools used on our network
- Cross-reference with Microsoft's risk assessment list
- Blacklist tools that pose security or compliance risks
- Track which tools are being used across the organization

**What this means for you:** You're free to use AI tools that aren't blocked — but we'd like to know about it.

---

## Before you start using a new AI tool

**Please contact us first.** Here's why:

1. **We can tell you if others are already using it** — you might find colleagues to share experiences with
2. **We can check if it's been risk-assessed** — save yourself potential problems
3. **We can add it to our overview** — helps us support everyone better
4. **We might have tips** — someone else may have figured out the best settings

- IT Security Architect Peter Frederik Larsen Koch: [pela@danishagro.dk](mailto:pela@danishagro.dk)

## Currently blocked tools

Tools on Microsoft's high-risk list are automatically blocked. If you try to access a blocked tool, you'll see a notification.

**If a tool you need is blocked:** Contact IT Security to discuss whether an exception is possible or if there's an approved alternative.

---

## Our recommended tool: Microsoft Copilot

Copilot is Danish Agro's official AI tool. It's included in our Microsoft 365 agreement with enterprise-grade security.

**Why Copilot is the safest choice:**

- Data stays within our Microsoft environment
- Enterprise-grade security and compliance
- No training on your data
- Integrated with your existing tools (Word, Outlook, Teams)

**Other tools (ChatGPT, Claude, Gemini, Le Chat):** Can be used for non-sensitive tasks, but remember to follow the Golden Rules — especially Rule 2 about confidential data.
